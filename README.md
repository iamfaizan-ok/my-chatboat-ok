# my-chatboat-ok


🧠 AI Chatbot System (Django + GPT Integration)

This project is a full-stack AI-powered chatbot system designed to deliver an interactive conversational experience with persistent chat history and user authentication.

It includes a clean and responsive frontend along with a robust backend powered by Django and Python, integrated with GPT through a structured API pipeline.

🚀 Features
🔐 User Authentication System (Login/Register)
💬 Real-time AI Chat Interface
🧾 Chat History Management (Saved Conversations)
🤖 GPT API Integration for Intelligent Responses
🌐 Responsive UI using HTML, CSS, and JavaScript
⚙️ Backend built with Django (Python)
🔄 API-based pipeline for seamless communication between frontend and AI model
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Django, Python
AI Integration: GPT API (via pipeline)
Database: Django ORM (SQLite / configurable)
📌 Project Overview

The system allows users to log in securely, interact with an AI chatbot in real-time, and access their previous conversations anytime. The backend handles authentication, message processing, and API communication with GPT, ensuring fast and intelligent responses.

This project demonstrates practical implementation of full-stack development combined with modern AI integration.
